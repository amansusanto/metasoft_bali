Dokumentasi API

Gunakan endpoint berikut untuk integrasi dengan aplikasi Anda.

POST /send

Mengirim pesan teks cepat.

{
  "to": "6281234567890",
  "message": "Halo, ini pesan otomatis."
}

POST /send-invoice

Mengirim nota penjualan dengan link PDF.

{
  "to": "6281234567890",
  "ref": "INV-001",
  "invoice_url": "https://example.com/nota.pdf"
}

POST /reminder

Mengirim pengingat tagihan pelanggan.

{
  "to": "6281234567890",
  "amount": 1000000,
  "due_date": "2025-10-15"
}

Gunakan header Authorization: Bearer <API_KEY> jika diperlukan oleh server Anda.
